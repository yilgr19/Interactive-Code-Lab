# Interactive Code Lab

Laboratorio web para aprendizaje de lógica de programación con teoría, cuestionarios, retos y (opcionalmente) modelos 3D / RA. Prototipo con usuarios y progreso en el navegador (`localStorage`).

## Instalación y ejecución

La guía completa (servidor local, **PC y móvil**, firewall, usuarios de prueba) está aquí:

**[INSTALACION.md](INSTALACION.md)**

En GitHub, al abrir el repositorio, el `README.md` se muestra debajo de los archivos; el enlace anterior lleva directamente a la guía en el mismo repo.

### Inicio rápido

```powershell
cd "ruta\al\proyecto"
python -m http.server 8000 --bind 0.0.0.0
```

- **PC:** [http://localhost:8000/login.html](http://localhost:8000/login.html)  
- **Móvil (misma Wi‑Fi):** `http://IP_DE_TU_PC:8000/login.html` — detalles en [INSTALACION.md](INSTALACION.md).
